# PodiumX Community Platform

## Overview
PodiumX Community is a social fitness matchmaking platform that connects training partners, enables session booking, and builds a healthy-living community. The platform combines Tinder-style matching with comprehensive fitness networking features.

## Brand Identity
- **Name:** PodiumX Community
- **Tagline:** Train Together. Elevate Together.
- **Colors:** Matte Black (#000000), Electric Green (#00FF5E), White (#FFFFFF)
- **Style:** Futuristic, Minimalist, Athletic, Tech-driven

## Core Features
1. **Smart Matching** - Swipe-based partner discovery
2. **Location Discovery** - Interactive map for gyms and training spots
3. **Session Booking** - Schedule joint training sessions
4. **Real-time Chat** - Secure messaging system
5. **Gamification** - Points, achievements, and leaderboards
6. **Community Events** - Local meetups and workshops

## Membership Tiers
- **Free:** Basic features, 10 daily swipes, 3 matches/day
- **PodiumX Plus ($19/month):** Unlimited matching, premium coaches
- **PodiumX Pro ($49/month):** Coach accounts, revenue sharing

## Tech Stack
- Frontend: React 18 + TailwindCSS
- Database: Trickle Database
- Icons: Lucide
- Fonts: Bebas Neue, Montserrat, Inter

## Project Structure
- `index.html` - Landing page
- `platform.html` - Authentication page
- `dashboard.html` - Main app dashboard
- `components/` - Reusable UI components
